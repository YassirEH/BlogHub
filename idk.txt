1. Blog CRUD Operations

Authenticated users can create blogs, optionally uploading an image. Blog data includes title, content, summary, tags, published status, and author.

Authenticated users can add comments to blogs.

Anyone can fetch comments for a blog.

Routes like creating, updating, deleting blogs, liking, and commenting are protected by JWT authentication.

Middleware ensures only the blog owner can update or delete their blog.
5. File Uploads

Blogs can have an image uploaded and stored on the server.
